# Boston University Masters in Data Science
## dx 699
dataset_exploration and dataset_exp2 cover data analysis I did for dx699. The purpose of the project was to observe whether STEM graduates were facing improving or depreciating job prospects in recent years. The data used is from the National Survey of College Graduates (NSCG) and the Survey of Doctoral Recepients (SDR). It includes univariate, bivariate, multivariate analyses, PCA, and random forest model building.
